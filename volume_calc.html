Volume Calculator Widget

Description:

A configurable widget you can use to calculate the volume of a space in cubic feet. We originally made it to calculate how much garden mix we needed for the raised beds for our Edible Schoolyard project.

Demo:

http://www.farmsense.net/demos/volume_calc.html

Using it:

The widget is available as a hosted file at http://www.mediamesis.net/widgets/volume_calc.min.js and is configurable by including json within the linking script tag.

For example, if you'd like the widget to appear in your right column. Simply add the following tag within your right column div or section:

<script src="http://www.farmsense.net/widgets/volume_calc.js">
{
"title":"Garden Mix Calculator",
"font":"Arial",
"width":"160px",
"padding":"40px 40px",
"margin":"10px 20px",
"bgcolor":"#efefef"
}
</script>

The script tag accepts title, font, width, margin, padding & background color parameters in standard CSS form, and the length, width & depth fields are used for calculating volume.
